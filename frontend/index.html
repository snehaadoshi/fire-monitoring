Help & User Guide

Learn how to use each part of the Early Fire Monitoring Dashboard effectively.

This dashboard displays live fire and environmental readings from multiple modules. Each block represents a sensor node showing temperature, gas, and smoke levels. Alerts appear automatically when readings exceed defined limits.

  • 🟢 Green – Normal, safe conditions.
  • 🟡 Yellow – Slightly elevated readings, keep watch.
  • 🟠 Orange – Warning, possible early fire indicators.
  • 🔴 Red – Critical alert! Fire risk detected.
  • Date Tab: Select a monitoring date to view logged sensor data.
  • Control Tab: Start or shutdown connected modules remotely.
  • Parameter Tab: View module-wise parameters and calculated risk metrics.
  • Dashboard Blocks: Click on any block to view full readings in the left panel.
  • Temperature Limit: Adjust threshold to define warning/danger zones.
  • Gas Sensitivity: Tune based on environment conditions.
  • Refresh Interval: Controls live data update rate.
  • Alert Sound: Enable or disable audible alerts.
  • ❌ No Data: Check CSV upload or sensor connectivity.
  • ⚠️ Wrong Colors: Verify threshold settings.
  • 🔇 No Alerts: Ensure browser sound is not muted.
